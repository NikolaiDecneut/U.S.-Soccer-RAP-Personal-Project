# US Soccer Referee Abuse Prevention Policy: Data Visualization Project

## Project Overview
This personal data visualization project explores the real-world impact and 
patterns of referee abuse in youth soccer following the implementation of U.S. 
Soccer's updated Referee Abuse Prevention Policy (RAP, Policy 531-9), which took 
effect March 2025 under the "Respect The Call" initiative.

The original version of this project aims to collect from two sources, with more 
possible later.

Data is collected from two primary sources:
- Washington State Referee Committee (WASRC)
- US Youth Soccer Boys National Championships — Salt Lake City & Provo, UT 
  (July 13–23, 2026)

## Objectives
- Visualize the frequency, type, and severity of reported referee abuse incidents
- Identify trends by age group, competition level, and referee years of experience
- Compare pre- and post-RAP policy data where and if available
- Highlight gaps in reporting and enforcement at the grassroots level, if possible

## Data Sources
| Source | Type | Access Method |
|---|---|---|
| Washington State Referee Committee | Incident reports, suspension records | Committee request / survey |
| USYS Boys Nationals 2026 (Utah) | Observational + survey data | On-site with admin approval |
| U.S. Soccer RAP Policy 531-9 | Policy/penalty matrix reference | Public (ussoccer.com/rap) |
| Washington Youth Soccer RAP Committee | Reported incidents | WYS Director of Operations |

## Variables of Interest
- Abuse type (verbal taunting, threatening language, physical contact, etc.)
- Sanction level (Level 1–4 non-physical; Level 1–3 physical)
- Referee age/experience level
- Match type (recreational, select, premier, ECNL, MLS Next)
- Reporter role (referee, coach, spectator, administrator)
- Geographic region(s) (WA state vs. national event)
